Welcome to Gappa

Gappa is a powerful Discord bot that helps you check if the music you want to use is copyrighted. Protect your content and avoid copyright strikes with ease!

How It Works

  1. Invite Gappa to your Discord server
  2. Use the command !check followed by the song name
  3. Gappa inspects the songs license, description and Title
  4. Receive instant feedback on the copyright status of the song

Features

  • Quick, Fast and FREE results
  • Highly demanded by the community

Top-Level Design

Our top-level design consists of the following components:

  • Discord Bot Interface: Handles user interactions within Discord
  • Command Parser: Interprets user commands and extracts song information
  • Engine Design: Designed for speed and efficiency
  • Response Generator: Creates user-friendly responses based on search results
